# electron-react-ts-starter

## USAGE

Put your react app under `src/renderer`,

Put your electron app under `src/main`,

## COMMANDS

#### Development

`npm run dev` or `yarn dev`

#### Build

`npm run build` or `yarn build`

#### Dist

`npm run dist` or `yarn dist`

> Only for mac (for now).

## CUSTOMIZATION

Edit `app.json`.
